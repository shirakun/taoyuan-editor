# taoyuan-editor
setube/taoyuan save editor

一个基于浏览器的加密存档编辑器工具，无需安装即可直接在本地修改游戏存档数据，支持属性、工具、武器、背包、农场等多维度编辑。

打开 https://shirakun.github.io/taoyuan-editor 上传存档即可进行编辑

## 功能特性

- 本地运行且加密
  - 纯前端页面，所有数据在浏览器本地读取与写入，不上传服务器。
  - 使用 AES 对存档内容进行加解密，默认密钥可自定义，提高安全性。

- 可视化存档管理
  - 拖拽或选择方式导入存档文件，自动识别并解密后进入编辑界面。
  - 分为多个标签页：玩家信息、工具、武器、背包、农场、Raw JSON 原始数据，方便快速定位需要修改的内容。

- 玩家与世界数据编辑
  - 玩家名称、性别、金钱、体力、体力上限、HP 等基础属性可直接修改。
  - 年份、季节、日期、时间、天气与明日天气等世界进度信息可同步调整。

- 工具与武器编辑
  - 一键将所有工具设置为基础/铁/钢/铱等不同等级，批量便捷强化。
  - 支持管理已拥有武器、装备中的武器以及附魔效果，并提供武器列表快捷添加。

- 背包与物品系统
  - 背包物品列表可编辑物品 ID、数量与品质，并内置完整物品名称搜索与筛选功能。
  - 支持快速添加新物品、删除物品，以及数量上限校验（如最大 99 堆叠）。

- 农场地块与作物
  - 网格化展示农田地块状态，显示作物、成长天数、是否浇水、是否杂草/虫害等。
  - 支持批量操作：一键设置所有作物成长、最大成熟、清除杂草与虫害、统一浇水等。

- 高级 JSON 编辑
  - Raw JSON 标签页直接显示完整存档 JSON 内容，可手动编辑后一键应用回界面。
  - 界面修改和 JSON 修改可以互相同步，方便高级用户进行精细控制。

## 安全说明

- 本工具不会向任何服务器上传存档内容，所有操作均在浏览器内完成。
- 加解密依赖用户提供的密钥，如忘记或错误密钥将无法正确解析存档，请妥善保存密钥。
- 建议在修改前手动备份原始存档文件，以便出现异常时可以恢复。
